Quick Setup & Run
-----------------

1.  **Install dependencies**
    
    `   pnpm install   `

2.  **Build the extension (if needed)**
    If you have TypeScript:

    `   pnpm run build   `

3.  **Run the extension in VS Code**
  *   Press F5 in VS CodeThis will open a new Extension Development Host window where you can test your Pac-Man game.
